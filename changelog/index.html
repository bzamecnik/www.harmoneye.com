Changelog and older releases

Here you can find older releases and summary of their changes.

The current release and installation instructions are available at the download page.

HarmonEye 1.4 (2014-01-22)

  • better harmonic tone detector
  • percussion filtering and enhancement of the harmonic components - far better visualization of tones
  • new app icon
  • modified coloring
  • accumulation mode uses exponential window (a moving window of samples) instead of moving average (all samples since some time)

HarmonEye 1.3 (2013-11-13)

  • major performance improvements

    • a better algorithm to compute the tones
    • blazing fast initialization
    • substantially less memory consumption
    • using our own sparse matrix implementation
    • different FFT library

  • greater percision - 7 bins bet tone instead of 5

  • another major code clean-up

HarmonEye 1.2 (2013-11-13)

  • licensing has been removed, HarmonEye is FREE since now!
  • a major code cleanup and refactoring
  • some minor performance enhancements
  • simpler DMG installer instead of PKG

HarmonEye 1.1.1 (2013-07-11)

  • fixed the trial mode

HarmonEye 1.1 (2013-07-01)

  • trial mode
  • performance optimizations

HarmonEye 1.0 (2013-06-04)

  • initial release
